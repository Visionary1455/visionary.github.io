搭建Github静态个人网站

利用Github的Github Pages功能搭建属于自己的静态个人网站

只需要准备一个自己喜欢的域名即可开始，可在腾讯云或者阿里云购买域名

以下教程操作是使用的腾讯云

第一步：新建一个Github的仓库

1.打开Github网站

2.点击右上角Sign Up注册一个新的帐号

3.注册完帐号后在主页右上角点+号

4.选择第一个选项New repository新建一个库

5.在Repository name填写仓库名，格式为http://username.github.io

6.点击下方Create repository成功创建我们的库

第二步：在Github库里上传静态HTML文件

上传自己写好的HMTL文件或可自行在网上找寻自己喜欢的HTML模版，也可以直接转载别人的库

点击 creating a new file 创作你的网站或者 uploading an existing file上传你已经创作好的网站

使用 Github Desktop 上传文件夹等多文件比较方便

以下教程为转移库

1.在新建的库我们会看到Quick Setup页面

2.把点击最下方的Import code

3.在框里填入库的URL

4.点击Begin Import开始转载

第三步：设置Github Pages

1.在我们刚刚创建好的库界面找到设置按键Settings

2.点击Settings后选择左边菜单的Pages

3.在Custom domain里输入你购买的域名

4.打开下面的Enforce HTTPS

到这里我们就设置好Github Pages了

第四步：在腾讯云或者阿里云解析域名

1.打开腾讯云域名管理页面

2.点击自己域名后面的解析功能

3.点击添加记录

4.这里我们需要使用CNAME记录，主机记录填写www，记录值填写刚刚我们创建好的Github库名字，然后点击确认

5.这个时候我们已经成功让域名和Github创建的库绑定

Github的服务器是在国外的，国内IP第一次访问需要长时间等待网页加载
